## 目录结构

```
src/
├── assets/            // 静态资源文件夹
├── api/            // 接口
│	├── request.js       // axios实例
│	└── character.js  // 角色相关接口
├── components/        // 可重用的组件（如角色卡片等）
├── views/             // 页面视图
│   ├── Home.vue       // 首页，展示所有角色
│   └── CharacterDetail.vue  // 角色详细信息页面
├── router/            // 路由配置
│   └── index.js       // 路由文件，定义首页和角色详细页的路由
└── App.vue            // 主组件
```
